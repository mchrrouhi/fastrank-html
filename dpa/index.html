Data Processing Agreement (DPA)

Effective date: November 21, 2025

This Data Processing Agreement (“DPA”) forms part of the agreement between the customer (“Controller”) and Cadunlock LLC (“Processor”, operating Fastrank), and governs the processing of personal data in accordance with GDPR and other applicable data protection laws.

1. Definitions

Key GDPR terms used in this DPA include:

  • “Controller”: The entity determining purposes of personal data processing.
  • “Processor”: The entity processing data on behalf of the Controller.
  • “Subprocessor”: Third parties engaged by the Processor.
  • “Personal Data”: Any information relating to an identified or identifiable person.

2. Roles of the Parties

For the purposes of providing website audits, optimization, and consulting services, the Controller determines the purpose and means of processing, while Cadunlock LLC acts strictly as a Processor.

3. Nature & Purpose of Processing

The Processor may process personal data for:

  • Technical audits & performance diagnostics
  • Delivering optimization services
  • Analyzing website performance metrics
  • Providing support or troubleshooting
  • Security monitoring & fraud prevention

4. Types of Personal Data Processed

  • Contact details (email, name, company)
  • Technical data (URLs, logs, performance metrics)
  • User analytics or event data
  • Limited billing metadata (from Stripe)

5. Controller Obligations

  • Provide lawful instructions for data processing
  • Ensure accuracy of provided data
  • Ensure lawful basis for processing

6. Processor Obligations

  • Process data only under Controller instructions
  • Maintain confidentiality
  • Implement appropriate technical and organizational measures (TOMs)
  • Assist the Controller with GDPR obligations
  • Notify of data breaches without undue delay

7. Security Measures

We implement industry-standard security practices including:

  • Encrypted storage and transport (HTTPS/TLS)
  • Access controls & authentication restrictions
  • Regular security reviews & monitoring
  • Secure data deletion procedures

8. Subprocessors

We use verified subprocessors such as:

  • Stripe (payments)
  • Crisp (support chat)
  • Hosting & analytics providers (e.g., Google Cloud)

All subprocessors are under written agreements ensuring GDPR-compliant protections.

9. International Data Transfers

When transferring personal data outside the EU/UK, the Processor uses:

  • Standard Contractual Clauses (SCCs)
  • Supplementary security measures
  • Data minimization & encryption

10. Data Breach Notification

In the event of a confirmed personal data breach, we will notify the Controller without undue delay, providing all available relevant information.

11. Data Subject Requests

The Processor assists the Controller in responding to GDPR data rights requests such as access, deletion, correction, or portability.

12. Data Return or Deletion

Upon termination of services, the Processor will delete or return all personal data unless legally required to retain it.

13. Duration of Processing

This DPA remains in effect as long as the Processor processes personal data on behalf of the Controller.
